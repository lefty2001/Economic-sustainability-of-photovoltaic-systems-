# Economic-sustainability-of-photovoltaic-systems-
This research aims to evaluate whether accurate day-ahead PV forecasts and explicit error prediction reduce imbalance penalties, with France data from ENTSO-E and Open-Meteo chosen as a case study. This work employs 6 models (Linear Regression, Random Forest, MLP, XGBoost, LSTM and CNN-LSTM) are compared against the ENTSO-E baseline.
